# Card Monsters Wiki

**The complete Card Monsters encyclopedia** — cards, passives, decks, and strategy tools, all available **offline** with built-in update tracking.

  Windows desktop app · Local database (`card.json`, `passives.json`, images, and audio)

---

## Home

Your hub for everything in the game:

- **Latest Arrivals** — the 5 newest cards showcased in an animated arena
- **Recently Added** — quick preview of the latest unique cards added to the database
- **Live stats** — total cards, passives, Elite, Legendary, Awaken cards, and more
- **News feed** — preview of announcements (new cards, deck builder updates, etc.)

- Actual version here : https://github.com/Sehla184649/cardmonsterwiki/releases

---

## Card Library

Browse the full game with powerful search tools:

- **Filters** — type (Monster, Equipment, Spell, Flash), rarity, faction, mana cost (visual chips), passive
- **Sort** — newest first, mana ascending/descending, HP / armor, alphabetical
- **Pagination** — 12 to 100 cards per page, or show all
- **Global search** — press `/` to find a card or passive from anywhere
- **Detail view** — click a card for stats, passive explanations, and an immersive typewriter effect
- **Right-click** — send a card straight to one of your **5 decks**, or add it to the comparator
- **View modes** — small or large card icons

> Tip: right-click a card → full details or send to a deck.

## Grimoire of Passives

A paginated grimoire to master every ability in the game:

- **Visual encyclopedia** — each passive with its hex icon and full description
- **Tier filters** — personal ranking S, A, B, C, D, or untiered (right-click to assign)
- **Search** — find a passive by name or keyword
- **Book navigation** — left / right pages to browse the encyclopedia
- **Card links** — right-click a passive to see every card that has it

## Deck Forge Pro

A strategic lab to build, analyze, and share your decks.

### Deck building

- **5 independent deck slots**, saved locally
- **8 monsters** (battalion) + **8 spells & equipment** — official game rules enforced
- **Built-in library** — mana, faction, type, and rarity filters; fullscreen mode available
- **Drag & drop** — add cards from the library or reorder your deck
- **Export / Import** — share a deck via a compact code
- **Smart validation** — Elite alerts (max 2 factions), equipment / monster color conflicts, copy limits

### Deck Intelligence

Real-time analysis panel as you build:

| Metric | What it shows |
|--------|----------------|
| **Detected archetype** | Suicide War, Magic Balance, Dice Fortune, hybrid… |
| **KPIs** | Average mana, ATK, HP, total ATK |
| **Faction Spread** | War / Balance / Fortune / Chaos / Nature breakdown |
| **Mana Curve** | Visual mana curve chart |
| **Combat Profile** | Summary of your deck's combat identity |
| **Attack Synergies** | Most common offensive passives |

### Strategic Tips — contextual advice

The analysis engine generates **personalized tips** based on your deck composition:

- **Magic Balance** — Magic Reflect, Magic Piercing, Amplify / Mantra / Arcane Power synergy
- **Suicide War** — Trample, Piercing, Charge + Amplify + Doublestrike combo
- **Dice Fortune** — Trickery, Power of Fate, Random Encounter
- **Color conflicts** — warning when equipment doesn't share a faction with your monsters
- **Composition triggers** — automatic tips, for example:
  - Melee + Ranged → use **Reach**
  - High mana curve → **Equilibrium** and crystal generation
  - All-physical battalion → stack **Inspire**, **Trample**, **Piercing**
  - Poison core → **Toxic Bomb**, **Poison Halo**
  - Low-cost swarm → watch out for **Contempt**
  - Growth package → Flash heals on full-HP creatures
  - Melee guide — reminders on Flying, Shield, Thorns, Counter-Attack, Opportunity, Demoralize

Each tip shows the **relevant passive icons** so you can instantly see what's missing.
